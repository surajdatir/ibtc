Get Started with Sahyadri

Sahyadri is a decentralized network built for everyone. Whether you want to save value, secure the network, or build applications, getting started takes only a few minutes. Choose your path below.

Path 1 — For Individuals

Step 1: Set Up Your Wallet
To interact with the Sahyadri network, you need a digital wallet to store your CSM. Your wallet generates a unique address where you can receive funds.
✧ Backup your seed phrase – 12–24 words that control your coins. Keep it offline.
Download CSMWallet (Win/macOS/Linux/Mobile)
Download Wallet Learn more

Path 2 — For Miners

Step 2: Start Mining with HeavyHash
Sahyadri is ASIC-resistant. You don't need expensive hardware; use your computer's CPU or GPU.
  • Download Sahyadri Node (full ledger sync)
  • Join a mining pool for steady rewards
  • Mobile mining (low-energy, modern smartphones)
    • Download Node Pool list

    Path 3 — For Developers

    Step 3: Integrate and Build
    Sahyadri provides a robust API and modular framework for payment gateways, identity solutions, and layer-2 protocols.
  • Read the Documentation (Core architecture)
  • Explore GitHub repository & contribute
    • Documentation GitHub

    The Golden Rules

    • Not Your Keys, Not Your Coins: Always keep your private keys/seed phrases offline.
    • Verify the URL: Always ensure you are on official Sahyadri website before downloading software.
    • Start Small: When sending CSM for the first time, try a small test transaction.
