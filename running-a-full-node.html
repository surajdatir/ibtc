Running a Full Node

Strengthen the Sahyadri Network

Running a full node helps maintain the security, decentralization, and transparency of the Sahyadri blockchain. Full nodes store the entire blockchain and independently verify all transactions and blocks.

By running a node, you help ensure that the network remains secure, reliable, and resistant to censorship.

What is a Full Node?

A full node is a computer that downloads and verifies the complete Sahyadri blockchain. Unlike lightweight wallets, a full node independently validates every transaction according to the network rules.

Full nodes help maintain the integrity of the network by ensuring that all blocks follow the protocol.

Why Run a Full Node?

Running a Sahyadri full node provides several benefits:

Network Security

Full nodes verify transactions and blocks, helping secure the blockchain from malicious activity.

Decentralization

More nodes mean a stronger and more decentralized network that cannot be controlled by any single entity.

Independent Verification

Full nodes allow users to verify transactions without relying on third parties.

Supporting the Ecosystem

Node operators play an essential role in keeping the Sahyadri network healthy and operational.

Basic Requirements

Before running a Sahyadri full node, ensure your system meets the following requirements:

  • Stable internet connection
  • Minimum 4 GB RAM
  • Multi-core processor recommended
  • At least 200 GB of free storage
  • Linux / Windows / macOS supported

How to Run a Node

Follow these general steps to run a Sahyadri full node:

  1. Download the Sahyadri node software from the official repository.
  2. Install the software on your system.
  3. Sync the full blockchain with the network.
  4. Keep the node online to help validate transactions and blocks.

Detailed setup instructions will be available in the official documentation.

Keep Your Node Online

For maximum benefit to the network, node operators are encouraged to keep their nodes running continuously. The more active nodes the network has, the stronger and more decentralized it becomes.

Contribute to Decentralization

Running a Sahyadri full node is one of the most impactful ways to support the ecosystem. Every node strengthens the network and helps maintain the principles of decentralization and transparency.

Thank you for supporting the Sahyadri network. Every node makes a difference.
